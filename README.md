# Logistic Regression Notebook

##  Overview
This project demonstrates the implementation and understanding of **Logistic Regression**, a fundamental machine learning algorithm used for **binary classification** problems.

The notebook walks through data preprocessing, model training, evaluation, and interpretation of results using Python.

##  Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
